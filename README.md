CourseManager is a Java application integrated with MongoDB, designed to manage courses through standard CRUD operations. The project follows a Test-Driven Development (TDD) methodology, with a focus on achieving 100% code coverage.

## Key Features

- Course management with basic CRUD operations
- MongoDB repository integration
- Test-Driven Development (TDD)
- 100% Code Coverage using SonarCloud and Coveralls
- Continuous Integration with GitHub Actions, testing on Java in multiple environments
